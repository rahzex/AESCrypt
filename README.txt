# AESCrypt
AESCrypt is Simple tool that use AES-256 to keep files secured.
Before using this tool the Java Cryptography Extension (JCE) Unlimited Strength Jurisdiction Policy Files for JDK/JRE 8 must be installed 
in the system which can be found at http://www.oracle.com/technetwork/java/javase/downloads/jce8-download-2133166.html
Otherwise The Illegal Key Size exception will occur.
